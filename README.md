# 🌱 **Sahayogi Kisaan – Empowering Karnataka’s Farmers with Digital Agriculture**  

*"Farming isn’t just an occupation; it’s the foundation of our nation. With real-time stock updates, easy access to agricultural resources, and seamless connectivity with rural societies, we are transforming the way farmers procure essential supplies."*  

---

## 🌍 **Vision – Revolutionizing Agricultural Resource Management**  

Farmers in Karnataka struggle with resource availability, often traveling long distances or relying on intermediaries to check stock levels. Uncertain supply chains and fluctuating stock impact farming efficiency, leading to losses.  

**Sahayogi Kisaan is here to change that.**  

🚀 This **AI-powered web platform** ensures farmers get real-time stock availability, access to crucial farming inputs, and digital support—right at their fingertips.

## 📌 **Getting Started – Simple & Quick Setup**  

🔹 **Step 1: Clone the Repository**  
```sh
git clone https://github.com/your-repo/sahayogi-kisaan.git  
cd sahayogi-kisaan
```  

🔹 **Step 2: Install Dependencies**  
```sh
npm install
```  

🔹 **Step 3: Configure Supabase & Firebase**  
- Set up authentication in Supabase.  
- Enable PostgreSQL with Row Level Security (RLS).  
- Add your **Supabase API keys** inside `/config/supabase.ts`.  

🔹 **Step 4: Start the Development Server**  
```sh
npm run dev  
```
Your **Sahayogi Kisaan** platform is now live at `http://localhost:3000`! 🌱
